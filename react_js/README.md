# 🚀 보일러플레이트 설정 가이드

## 🔥 Git 커밋 메시지 템플릿 설정

### ✅ 1. Git 저장소 초기화

```sh
git init
```

### ✅ 2. 커밋 메시지 템플릿 설정

```sh
git config commit.template .gitmessage.txt
```

### ✅ 3. Git 저장소 상태 확인

```sh
git status
```

### ✅ 4. 설정 확인

```sh
git config --get commit.template
```

💡 위 단계를 순서대로 실행하면 Git 커밋 메시지 템플릿이 정상적으로 적용됩니다!
